# SauceDemo Automation Testing Project

## Technologies Used
- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Maven
- Page Object Model (POM)
- WebDriverManager

## Project Structure
- src/main/java/pages - Page Object Model classes
- src/main/java/utilities - Base class and helper methods
- src/test/java/stepDefinitions - Cucumber step definitions
- src/test/java/runners - TestNG runner
- features - Cucumber feature files

## Test Scenarios
- Successful login with valid credentials
- Login with invalid credentials

## How to Run
1. Clone the repository
2. Open in Eclipse as Maven project
3. Run TestRunner.java as TestNG test
